What to do when you want to access your code on Git when you boot up:
1. "git status"
2. "git pull"


What to do when you want to add your own code or editted code:
1. "git add .""
2. "git commit -m "*Insert message here*""
3. "git push"


How to set up GitHub to login automatically
1. git config remote.origin.url https://{USERNAME}:{PASSWORD}@github.com/{USERNAME}/{REPONAME}.git
2. Remove all "{" or "}" and enter "" around your password
3. Exclude the password if you are concerned about security
